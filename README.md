# flora.is-a.dev placeholder site

This folder contains a minimal static page intended for a GitHub Pages repository.

Recommended repository name:

`sanzhangSZ.github.io`

Deployment steps:

1. Create a public GitHub repository named `sanzhangSZ.github.io`.
2. Upload the files from this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to deploy from the default branch root.
5. Wait until `https://sanzhangSZ.github.io` is online.
6. Then submit the `flora.is-a.dev` registration JSON from the sibling folder.
7. After the PR is merged, add `flora.is-a.dev` as the custom domain in the GitHub Pages settings.
